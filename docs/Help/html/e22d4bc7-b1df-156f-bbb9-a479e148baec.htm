CommonFieldstoDict Method

Used to help properly serialize a request object. Need to return a dictionary containing all properties that are part of this class when a property has been set by the user of the class instance (request object). There is a flag that tells if the user set each corresponding property The flag property is only used on optional properties. If there is a required property, it should always be added to the dictionary
