This class represents a FreeClimb Recording instance. Recording objects are created by the SDK whenever a recording instance would be returned by the API; these primarily come from QueuesRequester inside a FreeClimbClient instance.

This class represents the set of wrappers around the FreeClimb Recordings API. It provides methods to handle all the operations supported by the FreeClimb Recordings API.
