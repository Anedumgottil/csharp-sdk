FreeClimbRequest Class

A FreeClimbRequest represents the JSON object that is sent to the webhooks (voiceUrl, voiceFallbackUrl, etc.) of FreeClimb applications. All webhooks except the statusCallbackUrl will recieve a payload in this form.
